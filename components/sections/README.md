# Sections Components

This directory contains modular section components extracted from the main page for better organization and maintainability.

## Components

- **HeroSection**: Landing section with animated introduction and social links
- **AboutSection**: About me section with typewriter effect and resume download
- **SkillsSection**: Skills showcase with animated cards
- **ProjectsSection**: GitHub projects display with loading states
- **ContactSection**: Contact form and social links
- **SocialLinks**: Reusable social media links component

## Usage

```tsx
import { HeroSection, AboutSection, SkillsSection, ProjectsSection, ContactSection } from "@/components/sections";
```

All components maintain the original logic and styling from the main page.
